# CityFinder Pro

**CityFinder Pro** — это личный каталог полезных городских мест. Проект разработан в рамках обучения в Яндекс Лицее.

## Основная идея
Пользователь создаёт свою карту полезных мест в городе: дешёвые кафе, сервисы, акции, скрытые локации. Сервис не просто ищет места, а хранит личный каталог с оценками, заметками, статистикой посещений и рекомендациями. Есть бесплатная версия для базового использования и Pro-версия с расширенной аналитикой и экспортом данных в CSV.

## Применённые технологии
- **Backend:** Python, FastAPI
- **База данных:** SQLite + SQLAlchemy (ORM)
- **Аутентификация:** JWT-токены, хэширование паролей (passlib, bcrypt)
- **Frontend:** HTML, CSS, JavaScript (Vanilla), Bootstrap 5
- **Карты:** API Яндекс Карт
- **Управление зависимостями:** `requirements.txt`
- **Работа с файлами:** Загрузка изображений на сервер и скачивание CSV

## Ключевые возможности
1. Регистрация и авторизация пользователей (хранение хэшей паролей).
2. Интерактивная карта (API Яндекс Карт) — выбор места кликом и отображение сохраненных локаций.
3. Добавление мест (название, категория, адрес, рейтинг, цена, заметки, геолокация).
4. Загрузка фотографий для каждого места.
5. Фильтрация и статистика по категориям.
6. Разделение функционала на Free и Pro версии.
7. Экспорт базы мест в формате `.csv` (только для Pro).

## Как запустить проект локально

1. Склонируйте репозиторий:
   ```bash
   git clone <url_репозитория>
   cd <папка_проекта>
   ```

2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv venv
   # Для Windows:
   venv\Scripts\activate
   # Для Linux/Mac:
   source venv/bin/activate
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустите сервер FastAPI:
   ```bash
   uvicorn main:app --reload
   ```

5. Откройте приложение в браузере:
   Перейдите по адресу: [http://127.0.0.1:8000/frontend/index.html](http://127.0.0.1:8000/frontend/index.html)

*Примечание: База данных `cityfinder.db` и таблицы в ней создадутся автоматически при первом запуске сервера.*
